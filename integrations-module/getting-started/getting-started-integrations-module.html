Introduction

The Integrations module allows you to set up integrations with popular 3rd party systems without working with APIs. To access the Integrations module, click the icon in the left navigation in Studio:

The Integration module will provide links to set up the available integrations:

Setting up integrations

Currently there are two integrations available that allow you to set up a Google Drive and/or Dropbox folder for automatic ingestion. Brightcove will check these folders periodically (about every ten minutes) and automatically ingest any new videos that have been added to your account.

Details of setting up the integrations are provided in these topics:

  • Setting Up an Automatic Ingestion Folder on Dropbox
  • Setting Up an Automatic Ingestion Folder on Google Drive
